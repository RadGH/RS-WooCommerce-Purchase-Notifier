# RS WooCommerce Purchase Notifier (WordPress Plugin)

_Set up custom email notifications to be sent based on a WooCommerce purchase of a specified product._

- [View this plugin on Zingmap.com](https://zingmap.com/plugin/rs-woocommerce-purchase-notifier/)

![Plugin thumbnail](thumbnail-min.jpg)

## Description

This plugin allows you to add email addresses to individual products. When a customer purchases the 
product, an email notification is sent to the specified email addresses.
